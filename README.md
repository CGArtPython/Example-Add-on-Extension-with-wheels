# Example-Add-on-Extension-with-wheels
This is an example Blender 4.2 Extension add-on that uses Python wheels to bundle third-party dependencies 

# How to get the Python package wheels
On Windows run
* grab_wheels.bat

On Linux/macOS run
* grab_wheels.sh
